Para executar o sistema basta compilar tal projeto em uma IDE de C++, preferencialmente o CLion devido as referencias de todas as classes, que no caso são 5.
Logo em seguida, rode o executável e aplique os comandos para validação.

Obs: Para que as alterações nos HDs (arquivos textos respectivos) criados ocorram faz-se necessário dar o comando exit no cmd para sair do HD e retornar ao "Gerenciador"
